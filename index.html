Skill 开源

将完整能力安装到你的 AI 编程助手中

什么是 Skill？

Skill 是一个 Markdown 配置文件，定义了 AI 助手在特定场景下的行为规范。安装后，AI 会按照 Skill 中的流程和质量标准来生成绩效反馈。

# 安装到 Claude Code
~/.claude/skills/performance-feedback-generator/SKILL.md
# 安装到 Codex
~/.codex/skills/performance-feedback-generator/SKILL.md

使用方式

1 从 GitHub 下载 SKILL.md 文件
2 放入对应 AI 工具的 skills 目录
3 在对话中说"帮我写一份绩效反馈"即可触发
GitHub 仓库

技术架构

基于 AI Agent Skill 格式，跨平台兼容 Claude Code 和 Codex。数据本地处理，零服务器依赖。

设计理念

结构化输入 → 专业框架处理 → 结构化输出。内置 SMART 建议、2:1 正负比等 HR 最佳实践。

适用场景

季度评估、1:1 准备、晋升推荐、入职评估。支持正式、温和、直接三种语气风格。
